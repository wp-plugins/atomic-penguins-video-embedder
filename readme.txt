==Description==

The Atomic Penguins Video Embedder is a video embedder plugin that adds a widget option called "Atomic Video Embedder" in the widget list. You can place the embed code inside the widget.

==Installation==

1. Upload plugin.zip to wp/content/plugins directory
2. Activate the plugins through the 'Plugins' menu in Wordpress Admin
3. A new widget option will appear on the widets menu called "Atomic Video Embedder"
